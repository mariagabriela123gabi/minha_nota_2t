PANDAS

Animais fofos ou biblioteca de software

Em programação de computadores, pandas é uma biblioteca de software criada para a linguagem Python para manipulação e análise de dados.

Pandas podem se alimentar por até 14 horas por dia. Os ursos pandas podem defecar mais de 100 vezes ao dia. Os pandas apresentam uma espécie de falso polegar que ajudam, por exemplo, o animal a ter uma maior firmeza ao pegarem seus alimentos.
